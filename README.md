# 🎉 Birthday Surprise Website (P02)

A creative and interactive birthday surprise website built using HTML. This project features a beautiful slideshow, background music/video, and animated love message effects to make special moments unforgettable.

---

## ✨ Features

* 📸 Image slideshow (01.jpeg – 06.jpeg)
* 🎬 Background video integration (`bairan.mp4`)
* 💖 Love message animations
* 🎨 Clean and attractive UI
* 📱 Responsive design

---

## 🚀 Live Demo

👉 https://jayanti-bdey.netlify.app/

---

## 📁 Project Structure

* `index.html` – Main webpage
* `01.jpeg – 06.jpeg` – Slideshow images
* `bairan.mp4` – Background video/music

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (basic animations)

---

## 💡 How It Works

1. Website loads with a slideshow of images
2. Background video/music plays automatically
3. Animated love message appears
4. User enjoys an interactive birthday experience

---

## ⚙️ Customization

* Replace images (`01–06.jpeg`) with your own
* Change video file (`bairan.mp4`)
* Edit message text inside `index.html`
